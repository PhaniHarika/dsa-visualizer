# 🧠 DSA Algorithm Visualizer

An interactive visualizer for Data Structures & Algorithms built with React. Watch sorting, searching and graph traversal algorithms come to life step by step!

## 🔴 Live Demo
👉 [Click here to view the app](https://dsa-visualizer-xi-tawny.vercel.app)

## ✨ Algorithms Visualized
- **Sorting:** Bubble Sort, Merge Sort, Quick Sort
- **Searching:** Linear Search, Binary Search
- **Graphs:** BFS (Breadth First Search), DFS (Depth First Search)

## 🛠️ Features
- Step-by-step animation with speed control
- Adjustable array size
- Big-O time & space complexity for each algorithm
- Color-coded comparisons and highlights

## 🚀 Tech Stack
React • JavaScript • CSS-in-JS

## 💻 Run Locally
```bash
npm install
npm start
```
